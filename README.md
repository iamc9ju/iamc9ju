# Ittipol Botmoon

วิศวกรซอฟต์แวร์สาย Fullstack ที่มุ่งเน้นการออกแบบและพัฒนาระบบ Backend ระดับ Production โดยเฉพาะระบบที่ต้องการความถูกต้องสูง ความปลอดภัย และความเสถียรในระยะยาว

---

## เกี่ยวกับผม

ผมสนใจการพัฒนาระบบที่มีความซับซ้อนเชิงธุรกิจ เช่น ระบบการเงิน ระบบธนาคาร และแพลตฟอร์มที่เกี่ยวข้องกับข้อมูลสำคัญของผู้ใช้  
ให้ความสำคัญกับการออกแบบสถาปัตยกรรมที่รองรับการขยายตัว ความถูกต้องของ Business Logic และการป้องกันช่องโหว่ด้านความปลอดภัยในระดับ API

แนวคิดหลักในการทำงาน:
- Architecture-first mindset  
- Security by Design  
- Testable & Maintainable Code  
- Production-ready approach

## โครงการที่กำลังพัฒนา

- **Banking Core System (Go)**  
  ระบบบัญชีแบบ Double-entry พร้อม Ledger และ Transaction Processor ที่รองรับความถูกต้องระดับองค์กร

- **NutriGo Platform (Fullstack)**  
  แพลตฟอร์มให้คำปรึกษาด้านโภชนาการ พร้อมระบบสิทธิ์การเข้าถึงข้อมูลที่ปลอดภัย

- **Secure Auth Service**  
  ระบบยืนยันตัวตนและกำหนดสิทธิ์การเข้าถึง (RBAC) พร้อมป้องกัน Broken Object Level Authorization

## แนวทางระยะยาว

มุ่งพัฒนาความเชี่ยวชาญด้านการออกแบบระบบระดับองค์กร โดยเฉพาะในสายงานการเงินและธนาคาร  
ให้ความสำคัญกับความถูกต้อง ความปลอดภัย และความยั่งยืนของโค้ดในระยะยาว
